# Icon Creation Instructions

To add a custom icon to the executable:

1. Create or obtain a .ico file (256x256 recommended)
2. Name it "icon.ico"
3. Place it in the project root directory
4. Run the build script again

For free icon creation, you can use:
- https://favicon.io/favicon-converter/
- https://www.icoconverter.com/
- Or any online ICO converter

Recommended: Use a magnifying glass or brain icon to represent AI/NER functionality.
